# Playwright Automation Framework

## Overview

A production-style Playwright automation framework built with JavaScript to automate the Practice Software Testing application.

## Objectives

- UI Automation
- API Automation
- Page Object Model
- Playwright Fixtures
- Data-Driven Testing
- CI/CD using GitHub Actions

## Tech Stack

- Playwright
- JavaScript
- Node.js
- Git
- GitHub
